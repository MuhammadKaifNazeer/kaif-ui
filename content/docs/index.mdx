## FAQ
                
Some common questions you may encounter:

<Accordions>

<Accordion id="What is Kaif UI?" title="What is Kaif UI?">
Kaif UI is an open-source component library designed to help developers create stunning, accessible, and customizable user interfaces. It provides ready-to-use components, blocks, and templates built with modern technologies like Next.js, Tailwind CSS, Framer Motion, and ShadCN UI.
</Accordion>

<Accordion id="Is Kaif UI free and open source?" title="Is Kaif UI free and open source?">
Yes, Kaif UI is 100% free and open-source. You can use it in your projects without any cost, and you're welcome to contribute to its development.
</Accordion>

<Accordion id="What tech stack does Kaif UI use?" title="What tech stack does Kaif UI use?">
Kaif UI is built using the following technologies:

- **React**: For building user interfaces.
- **Next.js**: For server-side rendering and static site generation.
- **Tailwind CSS**: For styling with utility-first CSS.
- **Framer Motion**: For creating smooth animations.
- **ShadCN UI**: For additional UI components and design systems.
</Accordion>

<Accordion id="Is Kaif UI customizable?" title="Is Kaif UI customizable?">
Yes, all components in Kaif UI are designed to be customizable. You can easily adjust the styling using Tailwind CSS and modify animations using Framer Motion to fit your project's needs.
</Accordion>

<Accordion id="Do I need to install Kaif UI?" title="Do I need to install Kaif UI?">
No, Kaif UI is a copy-and-paste library. There’s no installation required. Simply copy the component's code from the documentation and paste it into your project to start using it.
</Accordion>

<Accordion id="Can I contribute to Kaif UI?" title="Can I contribute to Kaif UI?">
Absolutely! Kaif UI is open-source and community-driven. You can contribute by improving the existing components, adding new ones, fixing bugs, or enhancing the documentation. Check the [Contribution Guide](/docs/contribute) to get started.
</Accordion>

<Accordion id="How can I get support if I have issues?" title="How can I get support if I have issues?">
If you encounter any issues or need help, you have multiple options:

- Open an issue on the [GitHub repository](https://github.com/MuhammadKaifNazeer/kaif-ui).
- Join the [Discord server](https://discord.com/invite/yZpeqXgqNc) and ask the community for support.

We encourage you to reach out, and we're always happy to assist!
</Accordion>
